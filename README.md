# Global Horizon | リアルタイム世界ニュース

世界各国の主要メディアから最新ニュースをリアルタイムで取得し、各国の言語の原文で表示するプレミアムなニュース閲覧アプリです。

## 🌟 特徴

- **多言語・多国間対応**: 日本、アメリカ、イギリス、フランス、ドイツ、中国の最新ニュースを網羅。
- **標準設定のカスタマイズ**: 起動時に表示する国と、翻訳の対象となる標準言語を自由に設定可能。
- **自動翻訳（デュアル表示）**: 取得したニュースが標準言語と異なる場合、原文とともに翻訳文を自動的に表示。
- **原文表示**: 翻訳結果を確認しながら、それぞれの国の言語（日本語、英語、フランス語、ドイツ語、中国語）のニュアンスもそのまま楽しめます。
- **プレミアムなデザイン**: ダークモードを基調としたグラスモーフィズム・デザインを採用。
- **滑らかなユーザー体験**: 
  - GSAP による美しいフェードインアニメーション。
  - コンテンツ読み込み中のスケルトンローディング表示。
  - Lucide による洗練されたアイコン。
- **環境に優しい設計**: ビルドツール（Vite等）のバージョンによる問題を回避するため、スタンドアロンな HTML/CSS/JS 構成で実装。

## 🚀 使い方

### 1. アプリの起動

ブラウザのセキュリティ制限（CORSポリシー）により、HTMLファイルを直接開くとニュースの取得がブロックされます。以下のいずれかの方法でローカルサーバーを介して起動してください。

#### A. 起動スクリプトを使用する場合（推奨）
ターミナルで以下のコマンドを実行：
```bash
./start.sh
```
自動的にサーバーが立ち上がり、ブラウザでアプリが開きます。

#### B. npmコマンドを使用する場合
```bash
npm start
```
その後、ブラウザで `http://localhost:8080` にアクセスしてください。

### 2. APIキーと標準設定

本アプリはニュースの取得に [GNews API](https://gnews.io/) を使用しています。
1. [GNews](https://gnews.io/) で無料のアカウントを作成し、APIキーを取得してください。
2. アプリ右上の **設定（歯車）アイコン** をクリックし、以下を設定してください：
   - **GNews APIキー**: 取得したキーを入力。
   - **標準の国**: アプリ起動時に最初に表示される国を選択。
   - **翻訳用言語**: ニュースを翻訳して表示する際の言語を選択。
3. **「保存して更新」** をクリックすると設定が適用されます。
   - 設定はブラウザ（localStorage）に保存されます。
